
동적 바인딩

부모와 자식이 같은 메서드를 들고있으면 부모의 메서드를 무효화 시킴 (overriding)

a.hello();를 부를수 있지만 그 위에 메서드는 무효화됨.

즉 한국인 메서드를 불러온다는 이야기 ( 자식의 메서드에 접근할 수 있음 )


지구인 a = new 한국인(); 일때

지구인 hello();

아시아인 hello(); (추가 => 재 정의 했다고 함 지구인hello() 무효화)
ㄴ> 출력됨

한국인

----------------------

지구인 hello() (무효화)

아시아인 hello() (무효화)

한국인 hello() ( 출력됨 )
