걍 메서드 이름을 바꿔라

System.out.println();
-------------------------
System( 클래스 )
out ( )
println ( 메서드 )

--------------------------

얘는 오버로딩 되어있다.

왜쓰냐 ? 사용자가 같은 이름으로 편하게 사용하게 하기위해

한계점

질럿 드라군 싸움 붙이는걸로 알아볼거

해당 attack 메서드는 질럿이 드라군밖에 못때림

복사해서 위치만 바꾸면 안겹침 = 오버로딩

유닛이 두마리니까 문제가 없다. ( 최대 갯수가 2개라서)

리버 추가하면 메서드 4개 더 필요함 ( 경우의 수 증가 )

캐릭터가 20개라면 ? ( 경우의수가 매우 많아짐 ) == 유지보수 매우 힘듬.

 -> 메서드가 늘어날수록 유지보수 난이도가 급상승함.

그래서 추상화를 배워야함. ( 수천개가 되도 하나로 해결 가능 )
